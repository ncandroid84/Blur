# **Descripción**

Activa el efecto blur en el centro de control, barra de notificaciones y pantalla de bloqueo.

Solo funciona en HyperOs3, para otras versiones busque mis otros módulos. 

# **Nota importante**

Si estas usando Ksu o Ksunext3.0 es necesario antes de instalar el módulo, tener instalado el metamodulo de overlayfs para que funcione, ya que en estas versiones se quito el montaje de sistema.
