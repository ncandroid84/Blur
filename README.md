Módulo que activa blur en el centro ce control, barra de notificaciones y pantalla de bloqueo.
Funciona únicamente en HyperOs3.0 
Si estas usando Ksu o Ksunext3.0 es necesario antes de instalar el módulo, tener instalado el metamodulo de overlayfs para que funcione

Si compartís esto en algún grupo o canal, no olvides darme los créditos.
